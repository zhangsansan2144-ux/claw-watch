# claw-watch 快速上手(给小白)

> Mac 用户,不用懂代码。复制粘贴就行,15 分钟搞定,以后再也不用管。

---

## 第一步:确保你有这些

- ✅ **Mac 一台**
- ✅ **Google Chrome 浏览器** —— 没有就去 [google.com/chrome](https://www.google.com/chrome) 下载
- ⚠️ **飞书账号** —— 你想让通知发到哪个群,你得是那个群的成员

> 上面的没装齐?先装齐再往下走。

---

## 第二步:打开"终端"

按 `Cmd + 空格`,输入 `终端`(或 `Terminal`),回车打开。

会出现一个黑底白字的窗口。**所有命令都在这里粘贴 + 回车**。

---

## 第三步:粘这一条命令,回车

```bash
cd ~ && git clone https://github.com/zhangsansan2144-ux/claw-watch.git && cd claw-watch && ./setup.sh
```

> 📂 这会把 claw-watch 装在 `/Users/<你的用户名>/claw-watch/`,**固定位置,以后别移动它**(移动了 cron 定时任务就找不到了)。

会自动跑安装,大概 2-3 分钟(主要是下载浏览器内核 ~150MB)。

> 如果报"git: command not found":你的 Mac 没装 Xcode 命令行工具。终端弹窗会让你"安装",点"安装"等几分钟,装完再粘上面那条命令。
>
> 如果报"没找到 Python 3.11+":粘这条 `brew install python@3.12` 装一下,然后再粘上面那条。
>
> 如果报"brew: command not found":去 [brew.sh](https://brew.sh) 复制首页那条命令粘进终端,装完再回头装 Python。

---

## 第四步:回答 4 个问题(全部回车,默认 Yes)

装完后会问你 3 个 `[Y/n]` 问题,**直接回车就是同意**。下面解释每一步发生什么:

### Q1: `现在开始登录向导?[Y/n]` → 回车

会依次走 3 个登录步骤:

1. **即梦** —— 自动弹出**真的 Chrome 窗口**到即梦登录页。**你扫码登录**,自动检测到登录成功后会自动关闭。
2. **LibLib** —— 同上,弹真 Chrome,你扫码,自动关。
3. **飞书 webhook** —— 终端会让你**粘一个 URL**。这个 URL 怎么拿?看下面 ⬇️

#### 怎么拿飞书 webhook URL(只做一次)

1. 打开**飞书 App**(手机或电脑都行)
2. 找一个群(没有就**新建一个**,可以只有你自己)
3. 进群,点右上角 `⋯` → 设置 → **群机器人** → **添加机器人** → **自定义机器人**
4. 给机器人起个名(比如 `claw-watch`)→ 点"添加"
5. 弹出一个 URL,长这样:`https://open.feishu.cn/open-apis/bot/v2/hook/xxxxxxxxxxxx`
6. **复制这个 URL**,回到终端粘进去 → 回车
7. 立刻会发一张 ✅ 绿色"配置成功"卡片到群里 → 你看到了 = 配好了

> 如果让你选"安全设置",**全不选**(留默认)就行。

### Q2: `装定时任务?周二+周五 11:00 自动跑+推飞书 [Y/n]` → 回车

写一行到你的 crontab,以后周二和周五的早上 11:00,Mac 会自动跑一次抓取并推送到你的飞书群。

> 你 Mac 必须**开机**才会跑(休眠也行,合上盖子那种"睡眠"也行,完全关机就不行)。

### Q3: `首次手动跑一次?[Y/n]` → 回车

立刻跑一次,作用:
- 测一下整套流程通不通(看飞书群是不是又收到一张卡片)
- macOS 可能会弹一个对话框 "是否允许 xxx 控制 Google Chrome" —— **必须点"允许"**,以后定时跑才不会卡

## 完事!

你会看到一行 `✓ 全部完成`。

之后**啥都不用管**:

- 📅 **周二 + 周五 北京时间 11:00**,飞书群里自动收到一张卡片
- 🆕 各个 AI 平台(可灵 / 即梦 / 海螺 / LibLib / TapNow / Lovart / Runway / Vidu / 拍我)有新功能、新公告就显示在卡片里
- ✅ 没新东西的日子也会收到一张绿色"各源平稳"卡片 —— 证明监控真跑过、你的账号还没掉线

---

## 极少数情况:坏了怎么办

| 情况 | 怎么办 |
|---|---|
| 周二 11 点啥都没收到 | 打开终端粘 `cd ~/claw-watch && tail data/cron.log`,把红色错误信息发给我 |
| 卡片里写"登录态已过期" | 粘 `cd ~/claw-watch && .venv/bin/claw-watch login`,走一遍向导,过期的那个会让你重新登 |
| 想换飞书群 / 换 webhook URL | 粘 `cd ~/claw-watch && .venv/bin/claw-watch login`,前 3 步全按 `s` 跳过,第 4 步重新粘新 URL |
| 想停掉定时任务 | 粘 `crontab -e`,找到带 `claw-watch` 的那一行,删掉 |
| 完全卸载 | 先 `crontab -e` 删行,再 `rm -rf ~/claw-watch` |
| 其他奇怪问题 | 微信问 `<你的微信>` |
